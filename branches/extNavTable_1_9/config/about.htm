Extensión NavTable v0.6

Web del proyecto: http://navtable.forge.osor.eu/

Descripción: :

Extensión para la visualización, navegación y edición de tablas de datos de capas vectoriales. Esta versión de NavTable es compatible con gvSIG-1.9.

Características:

  • Visualización y edición cómoda de valores alfanuméricos
  • Barra de navegación entre elementos
  • Configuraciones de selección y zoom
  • Cálculo de área y longitud de las geometrías
  • Nombres de atributos largos en Shapefiles
  • Copiado automático de datos del registro seleccionado
  • Creación y eliminación de registros
